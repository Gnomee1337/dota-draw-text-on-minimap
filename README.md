# Dota Draw Text On Minimap

Tool to avoid low-behavior chat mute in dota 2 by drawing text on minimap

<img src="https://i.imgur.com/ARUD2qE.jpeg"/>

# Setup:
```
$ pip install -r requirements.txt
```

# Usage:
```
$ python draw-on-minimap.py
```
